# 🌀 A_MAZE_ING

> 42 Urduliz — Grade: **125% ✅** (with bonus)

A collaborative maze generator built in Python, running entirely in the terminal.  
Generate, explore, and solve mazes — with a fully interactive menu.

---

## ✨ Features

- 🔀 **Two generation algorithms** — DFS (Depth-First Search) and Prim's algorithm
- 📐 **Configurable maze size** — width and height set by the user
- 🧭 **Interactive terminal menu** including:
  - Show / hide the solution
  - Change colors
  - Regenerate a new maze
- 🖥️ Full **terminal rendering** with clean visual output

---

## 🧠 Algorithms

### DFS — Depth-First Search
Generates mazes with long, winding corridors. Uses a recursive backtracking approach — carves a path as deep as possible before backtracking to explore other branches.

### Prim's Algorithm
Generates mazes with a more uniform, branching structure. Starts from a random cell and grows the maze by randomly adding neighboring walls — producing a different aesthetic and texture than DFS.

---

## 🚀 Usage

```bash
git clone https://github.com/PaulPouss/A_MAZE_ING
cd A_MAZE_ING
python3 main.py
```

Follow the interactive menu to choose your algorithm, set the maze size, and explore.
